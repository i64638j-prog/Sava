## Описание программы
Expense Tracker - это графическое приложение на Python для отслеживания личных расходов. Программа позволяет:

- Добавлять расходы с указанием суммы, категории и даты
- Просматривать все расходы в удобной табличной форме
- Фильтровать расходы по категории и периоду дат
- Автоматически подсчитывать сумму расходов за выбранный период
- Сохранять данные в JSON-файл и загружать из него
- Проверять корректность вводимых данных

## Требования и установка зависимостей
- Python 3.6 или выше
- Стандартные библиотеки Python (tkinter, json, datetime)

**Установка дополнительных зависимостей не требуется**, так как используются только встроенные модули Python.

Проверьте, что tkinter установлен:
- **Windows/Mac**: обычно идет в комплекте с Python
- **Linux**: может потребоваться установка:
  ```bash
  sudo apt-get install python3-tk  # Ubuntu/Debian
  sudo dnf install python3-tkinter # Fedora
